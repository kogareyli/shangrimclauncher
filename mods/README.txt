Ajoute ici tes fichiers .jar de mods.

Pour synchroniser les mods vers le dossier de jeu (.shangrimc/mods/),
va dans Paramètres → "Synchroniser les mods" dans le launcher.

Les mods seront copiés automatiquement avant chaque lancement si tu actives
la synchronisation automatique.

Mods compatibles : Forge 1.20.1
